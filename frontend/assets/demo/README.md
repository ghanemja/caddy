# Demo Images

This directory contains demo reference images that are preloaded in the UI.

## Mars Rover Demo

To add the Mars rover demo image:

1. Save the Mars rover image as `mars_rover.jpg` in this directory
2. The image will automatically load when you open the Code Generation section
3. You can also click the "Load Demo" button to reload it

The demo image shows a Mars rover with 6 wheels (3 per side) and will be used as a reference for generating code to add wheels to the assembly.
